Блоки, которые есть на сайте:

1. Шапка
2. Контент (в центральной части страницы)
3. Подвал
4. Сайтбар (слева)

Дополнительный элемент

В ШАПКЕ находится панель, на которой расположены:
* ссылки для создания обсуждения
* для создания учетной запись 
* для входа в систему в свою уже созданную учетную запись.

В блоке КОНТЕНТ находится: 
* строка Поиска информации
* вкладка Заглавной страницы
* вкладка Страницы с обсуждениями
* ссылка Еще с выпадающим меню для просмотра кода и истории
изменений данной страницы. 

В блоке Контент также размещается: 
текстовый и фото контент, актуальный на сегодняшний день.

Основной контект размещен в подблоках: 

* избранная статья
* хорошая статья
* изображение дня
* новые материалы
* новости
* праздники дня
* совместная работа недели
* сотрудничество с другими проектами


В ПОДВАЛЕ размещена контактная и правовая информация.

В САЙТБАРЕ слева размещена:
* панель навигации по странице
* блок со ссылками для участия в данном проекте
* блок со ссылками для вывода информации на печать в разных форматах
* блок для перевода страницы на другие языки 
и т.д
